# JavProjects

A collection of small Java console applications / sample projects.  

## Projects Included

| File | Description |
|------|-------------|
| `Calculator.java` | A simple calculator that performs basic arithmetic operations. |
| `BmiCalc.java` | Calculates Body Mass Index (BMI) given user input. |
| `RockPaperScissorsGame.java` | Implements the classic Rock-Paper-Scissors game in Java. |

## Getting Started

To compile and run any of these programs:

```bash
# compile
javac *.java

# run (example)
java Calculator
